# 第三个功能：旅游博客辅助创作模块设计

## 1. 模块目标

旅游博客辅助创作模块用于根据用户提供的旅行素材，生成游记初稿、社交平台文案、标题建议和标签摘要。

## 2. 页面范围

### 2.1 素材填写页

路径：`/blog/create`

主要字段：

- 标题
- 目的地
- 开始日期
- 结束日期
- 人数
- 行程记录
- 美食记录
- 照片描述
- 消费摘要
- 个人感受
- 生成类型
- 写作风格

### 2.2 生成结果页

路径：`/blog/result/:id`

展示内容：

- 生成标题
- 标签
- 正文
- 风险提示
- 生成时间

### 2.3 历史记录页

路径：`/blog/history`

展示内容：

- 素材标题
- 目的地
- 内容类型
- 写作风格
- 生成标题
- 创建时间
- 查看 / 删除操作

## 3. 数据表

### 3.1 blog_materials

保存用户输入的旅行素材。

### 3.2 blog_generations

保存每次生成的游记、文案、标题和标签。

## 4. 枚举值

### 4.1 content_type

- blog：游记初稿
- social_post：社交平台文案
- title_tags：标题和标签

### 4.2 writing_style

- guide：攻略型
- story：故事型
- casual：轻松分享型
- promotion：种草型

## 5. MVP 边界

首版只支持用户手动输入旅行素材，不接入真实图片识别。图片相关内容先通过“照片描述”字段录入。后续可以接入 VLM 实现图片理解。
